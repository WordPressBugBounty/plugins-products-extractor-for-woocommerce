افزونه رسمی ترب برای افزودن قابلیت‌های ترب به فروشگاه‌های ووکامرسی.

== Description ==

این افزونه برای فروشگاه‌های وردپرسی که از ووکامرس استفاده می‌کنند توسعه داده شده است تا قابلیت‌های موردنیاز ترب را به فروشگاه اضافه کند و یکپارچگیصف انتظار وبهوک محصول فروشگاه با خدمات ترب را فراهم سازد.

= نسخه‌های پشتیبانی‌شده =

* حداقل نسخه وردپرس: 5.9
* حداقل نسخه ووکامرس: 6.9

= نکات مهم =

* ووکامرس باید نصب و فعال باشد؛ در غیر این صورت افزونه اجرا نمی‌شود.
* حداقل نسخه‌های موردنیاز، وردپرس 5.9 و ووکامرس 6.9 هستند.
* پیوندهای یکتا نباید روی حالت Plain یا ساده باشند. از مسیر `تنظیمات > پیوندهای یکتا` یکی از حالت‌های دیگر را انتخاب کنید.
* درگاه وضعیت سفارش‌ها به صورت پیش‌فرض فعال است و از مسیر `تنظیمات ترب` قابل مدیریت است.
* پس از نصب و فعال‌سازی افزونه، برای تکمیل راه‌اندازی با پشتیبانی ترب هماهنگ کنید.

== Installation ==

1. مطمئن شوید وردپرس، ووکامرس و تنظیمات پیوند یکتا روی سایت شما فعال و آماده هستند.
2. افزونه را از پیشخوان وردپرس نصب کنید یا فایل آن را در مسیر `/wp-content/plugins/` قرار دهید.
3. افزونه را از بخش `افزونه‌ها` فعال کنید.
